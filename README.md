# GTA Scheduler

GTA Scheduler — десктоп‑приложение для автоматизации отправки предварительно настроенных сообщений в чат GTA 5 по расписанию с интеграцией Discord для авторизации и контроля доступа.

## Основные возможности

- Планирование последовательностей сообщений по точному серверному времени.
- Автоматическая активация окна GTA 5 и отправка команд в игровой чат.
- Гибкие задержки между сообщениями и шагами внутри последовательности.
- Создание скриншотов после выбранных сообщений.
- Авторизация через Discord с проверкой наличия на нужном сервере и ролей.
- Ограниченный доступ: приложение предназначено только для участников определённого Discord‑сервера.

## Технологии

- .NET / WPF для клиентского приложения.
- MaterialDesignInXaml для UI‑компонентов.
- Интеграция с Discord (OAuth2 / боты / API).
- Низкоуровневая работа с окном GTA 5.

## Установка и запуск

1. Получите актуальную сборку у администратора проекта или через приватный канал Discord.
2. Распакуйте архив в отдельную папку, где у пользователя есть права записи.
3. Запустите `GTAScheduler.exe` **от имени администратора** (необходимо для работы с окном GTA 5).
4. Авторизуйтесь через Discord в появившемся окне.
5. Настройте последовательности и расписание, затем запустите планировщик.

## Требования

- Windows 10/11.
- Установленный .NET Desktop Runtime (версия см. в релиз‑нотах).
- Установленная GTA 5 (игра должна быть запущена во время работы планировщика).
- Интернет‑соединение для авторизации через Discord.
